1.2.0

This minor release includes:

  • Online Multiplayer!
    • Still in early development
    • Limited to two person games
    • A little confusing to set up (a guide will be created soon)

I'm releasing online multiplayer in it's unfinished state. I plan to refine/complete it and when done, I will release version 2.0.0.

Multiplayer functionality depends on 'ngrok', a tool used for exposing your locally running server to others who want to join. In order to make requests to this server, they need to be authenticated with an ngrok auth token. Eventually I will include a way in the UI to input an auth token to attach to requests, but for now you will need to add your auth token directly in the code.

Add your ngrok auth token in matchmaking.js on line 21. You can either retrieve and use your own auth token (requires signing up for an ngrok account), or reach out to me and I can lend you mine.
